## zsh prompt (with IP)

Below will create a prompt with the IP address of the host eth0, or, if on VPN tun0. This can replace the standard `configure_prompt` method in Kali's default `.zshrc` file.

`(<user>ã‰¿<envrion>)-(<IP>)-[<pwd>]`
